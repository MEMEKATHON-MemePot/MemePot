# memepot

1. 신뢰
   DeFi이 프로젝트들은 신뢰를 바탕으로 활동해야합니다. 그래서 저희는 과장된 %가 아닌 현실성 있는 %로 장기적인 모델과 신뢰를 구축하고 있습니다.

2. 안전
   DeFi 프로젝트들의 가장 최우선은 자금 탈취 등 안전에 대비해야 하는 것을 최우선으로 해야 하는 것이라 생각합니다. 그래서 저희는 안전을 최우선적인 목표로 생각합니다.

3. 재미
   24h 365days 열리는 이벤트 풀로 상시 이벤트 개념의 재미와 감동을 드리는 것을 재미의 목표로 합니다.

4. 밈팟
   신뢰, 안전 목표이자 사명으로 생각하고 더 나아가 재미적 요소까지도 드리고자합니다. 밈팟은 단순히 디파이가 밈코어 재단의 가치를 계승하여 더 넓게 문화의 하나로써 자리잡으려고 합니다.
